
## Bash script for adding MongoDB/PostgreSQL dumps

## Usage

```bash
./loadDump.sh path/to/.env [-p postgres_dump_file] [-m mongo_dump_files] [-h]
```
## PostgreSQL env file:
```
POSTGRES_PASSWORD=
POSTGRES_HOST=
POSTGRES_USER=
POSTGRES_DB=
POSTGRES_PORT=
```

## MongoDB enf file:
```
MONGO_HOST=
MONGO_PORT=
MONGO_INITDB_ROOT_USERNAME=
MONGO_INITDB_ROOT_PASSWORD=
MONGO_AUTH_DB=
MONGO_INITDB_DATABASE=
```
